# MD Progress Tracking

Simple system for tracking progress in the Managing Director role.

## Structure

```
/tracking/md-progress/
├── README.md              # This file
├── progress-tracker.md    # Cumulative wins and timeline
├── weekly-logs/           # Weekly progress entries
│   └── log-YYYY-MM-DD.md  # Named by Monday date
└── communications/        # CEO/board update drafts
```

## Weekly Workflow

Each Friday (or end of week):
1. Create new log: `weekly-logs/log-YYYY-MM-DD.md`
2. List MD progress items (brief bullets)
3. List CTO progress items if relevant
4. Update `progress-tracker.md` with key wins

## File Naming

- Weekly logs: `log-YYYY-MM-DD.md` (Monday's date)
- Communications: `update-YYYY-MM-DD.md`
