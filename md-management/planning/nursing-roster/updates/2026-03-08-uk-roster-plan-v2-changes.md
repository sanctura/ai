# UK Roster Plan — V1 to V2 Changes

> Summary of changes made on 2026-03-08 incorporating CEO feedback.

## What Changed

| Area | V1 | V2 |
|------|----|----|
| Clinic hours | 07:00–19:00 (12hrs) | 08:00–20:00 (12hrs) |
| Opening | 07:00 (early nurse) | 08:00 (early nurse) |
| Closing | 19:00 (late nurse) | 20:00 (late nurse) |
| Peak hours | Not defined | 10:00–14:00 and 18:00–20:00 |
| Peak coverage | Incidental | Deliberate — double nurse during peaks |
| Maria Vega | Not referenced | Emergency floor backup (not scheduled) |
| Cecil (locum) | Gap-fill only | Gap-fill + planned peak-evening cover |

## Why

James reviewed V1 and provided the following feedback:
1. **Clinic hours**: Should be 08:00 (or 08:30) to 20:00 for the UK clinic
2. **Peak hours**: 10:00–14:00 and 18:00–20:00 need strong coverage
3. **Emergency backup**: Need overlap/backup — if there's an emergency on the floor, we need double coverage

## How Each Point Was Addressed

### 1. Clinic Hours (08:00–20:00)
All three shifts adjusted forward by 1 hour:
- Early: 07:00–16:00 → **08:00–17:00**
- Late: 11:00–19:00 → **12:00–20:00**
- Anchor: 09:00–17:00 → **10:00–19:00**

### 2. Peak Hour Coverage
Shifts staggered to maximise overlap during peaks:

| Time Slot | Nurses on Floor | Peak? |
|-----------|----------------|-------|
| 08:00–10:00 | 1 | |
| 10:00–12:00 | 2 | **Morning peak starts** |
| 12:00–17:00 | 3 | **Full peak 10:00–14:00** |
| 17:00–19:00 | 2 | |
| 19:00–20:00 | 1 | Evening wind-down |

- **10:00–14:00 peak**: 2–3 nurses throughout. Fully covered.
- **18:00–20:00 peak**: 2 nurses (18:00–19:00), 1 nurse (19:00–20:00). Cecil (locum) available for high-volume evenings.

### 3. Emergency Floor Backup
- **Maria Vega**: Available as emergency backup — not fixed hours. If a nurse is with a patient and gets called to a floor emergency, Maria V can step in. This is occasional, not scheduled.
- **Cecil (locum)**: Expanded role to include planned peak-evening cover on busy days, ensuring double coverage through to 20:00 when needed.

## What Didn't Change
- 3-week rotating cycle (Jasmin, Maria, Ali)
- Saturday on-call model (auto-forward, failover, voicemail)
- Compensatory half-day Friday for anchor week
- Evening protected time for protocol study
- Reception roster (separate track)
- On-call compensation (deferred)
