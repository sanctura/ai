## Recommended Saturday On-Call Phone Model

### 1. Primary Structure

Clinic landline → Mobile diversion, 08:00–12:00 Saturdays

- Main clinic number auto-forwards to: Primary on-call nurse mobile
- Nurse supported by: On-call doctor (escalation only)
- 12:00 onwards: Line reverts to voicemail with clear instructions

### 2. Backup Safety Net

- If primary nurse does not answer within 20–30 seconds: Call auto-rolls to Saturday standby nurse
- If still unanswered: Voicemail triggers + SMS alert to primary nurse
- This prevents missed calls without over-staffing.

### 3. Voicemail Script (Premium, Reassuring)

> "You've reached the Sanctura on-call service. A nurse is available this morning for urgent clinical queries. If your call is not answered immediately, please leave a message and you will be called back shortly. If this is a medical emergency, please attend your nearest emergency facility."
