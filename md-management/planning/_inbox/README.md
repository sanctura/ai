# Planning Inbox

Drop zone for raw planning information, initiative updates, and new ideas.

## How to Use

1. Drop files here named `YYYY-MM-DD-description.md`
2. Say **"process planning inbox"** to trigger the planning-processor agent
3. The agent will:
   - Identify which initiative each file relates to
   - Route content to the correct initiative folder (info-base updates, supporting docs, ideas, updates)
   - Create a new initiative folder from `_TEMPLATE/` if needed
   - Move processed files to `_processed/`
   - Report back what was processed

## File Naming

- `2026-03-07-incentive-trial-week1-results.md`
- `2026-03-07-new-nursing-roster-idea.md`
- `2026-03-07-lead-management-proposal.md`

## What Goes Here

- Initiative status updates
- New initiative proposals
- Decision outcomes
- Meeting notes related to planning initiatives
- Feedback or ideas for existing initiatives
- Any planning-related content you want routed to the right place
